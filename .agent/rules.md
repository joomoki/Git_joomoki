# 에이전트 작업 규칙 (Agent Rules)

## 1. 언어 및 작성 규칙
- **모든 워크로그, 계획서, 아티팩트(Artifacts)는 반드시 `한글`로 작성해야 합니다.**
- 코드 주석 및 커밋 메시지 또한 가능한 한글로 작성하거나, 명확한 이해를 돕기 위해 한글 설명을 포함합니다.
- 사용자와의 대화는 항상 정중한 한국어로 진행합니다.

## 2. 산출물 관리
- `task.md`, `implementation_plan.md`, `walkthrough.md` 등의 모든 아티팩트는 한국어로 내용을 작성합니다.

## 3. 진행중 표현
- 진행완료는 녹색 체크박스, 백그라운드 진행중은 빨간색 체크박스로 알려줘

## 4. 실행 모드 (Auto-Execution)
- **작업 요청 시, 가능한 한 중간 멈춤 없이 최종 목표 달성 시까지 자동으로 실행합니다.**
- 사용자의 확인이 반드시 필요한 경우(치명적인 데이터 삭제, 금전적 비용 발생 등)를 제외하고는 스스로 판단하여 진행합니다.
- 오류 발생 시 스스로 해결을 시도하고, 해결 불가능한 경우에만 보고합니다.
