시스템 정보 & 데이터 출처

데이터 수집

본 서비스는 Python 백그라운드 스크립트가 24시간 가동되며 실시간에 가깝게 데이터를 수집합니다.

• 국내 주식: Naver Finance (크롤링) / KIS API

AI 점수 산출 알고리즘

기술적/기본적 분석 지표를 종합하여 0~100점의 투자 매력도를 산출합니다.

• 기술적 분석: RSI, 이동평균선(MA) 수렴/확산, MACD, 볼린저 밴드 위치 • 기본적 분석: PER, PBR, EPS 성장률 가중치 적용

시스템 구조

데이터는 PostgreSQL DB에 저장되며, 주기적으로 정적 웹 파일(JS)로 변환되어 배포됩니다.

사용자는 별도의 서버 요청 대기 시간 없이, 최적화된 정적 파일을 통해 즉시 데이터를 조회할 수 있습니다.

현재 시스템 상태: 정상 가동 중 (데이터 자동 갱신)
